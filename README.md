1.3. Тема проекта
Расчёт массы яблока с использованием технологий машинного зрения
1.4. Цель проекта
Разработать приложение, которое с помощью методов компьютерного зрения определяет массу яблока по его фотографии.
1.5. Основные задачи

Реализация алгоритма обнаружения и сегментации яблока на изображении.
Разработка методики оценки массы на основе геометрических параметров (размеры, форма).
Создание удобного интерфейса для загрузки фото и вывода результатов.
Тестирование и оптимизация модели на разнообразных изображениях яблок (здоровых и повреждённых).

1.6. Аннотация
Проект реализует автоматизированное определение массы яблока по фотографии с применением модели YOLOv8. Система способна различать здоровые и повреждённые плоды (классы apple и damaged_apple). Это решение упрощает контроль качества продукции в сельском хозяйстве, сортировку урожая и оценку товарного вида.

2.6. Примечания

Модель best.pt обучена на датасете с двумя классами: apple (здоровое) и damaged_apple (повреждённое).
Для наилучшей точности используйте фотографии одного яблока на светлом однородном фоне.
Расчёт массы основан на геометрических параметрах (площадь, диаметр) и калибровочной формуле.
